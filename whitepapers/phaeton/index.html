Phaeton – Slim Laden

Slim EV-laden op basis van PV-overschot en prijsprikkels, volledig lokaal gestuurd op Victron Venus OS.

PV-Aware Dynamic Tariffs Local First

Waarom Phaeton?

Veel EV-laders negeren zonne-energie en dynamische prijzen. Phaeton brengt energie- en prijsdata samen, zodat laden slimmer en betaalbaarder wordt zonder cloud-afhankelijkheid.

  • PV-aware: Laadvermogen volgt de zon.
  • Prijsbewust: Integreert Tibber dagprijzen.
  • Victron Native: Volledige D-Bus integratie.

De Uitdaging

"Huiseigenaren willen laden wanneer stroom goedkoop of overvloedig is. Phaeton vervangt handmatig schakelen door een deterministische control loop op de Victron Cerbo."

Functionaliteit

Control Loop

Berekent veilige setpoints op basis van grid limits en PV excess.

D-Bus API

Exposeert status voor Victron dashboards en remote console.

Rust Core

Veilige, snelle binary zonder runtime overhead.

Systeem Architectuur

Modules
  • Driver: Modbus TCP naar Alfen lader.
  • Tibber: Prijzen ophalen voor planning.
  • Web: Lokale Axum interface voor config.
Deployment
  • Target: Linux ARMv7 (Cerbo GX / Raspberry Pi).
  • Updates: Self-updating binary.
  • Config: YAML-based setup.

Aan de slag met Phaeton
